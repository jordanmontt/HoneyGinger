## Installation

To install in Pharo 7, execute the following lines in a playground:
```
Metacello new
    baseline: 'HoneyGinger';
    repository: 'github://tomooda/HoneyGinger/repository';
    load.
```
For demos, see the class-side methods of class `HGSimulator`.
